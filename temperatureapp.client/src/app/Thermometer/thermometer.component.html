About this application.

This application is created using Angular 16 and Syncfusion library for the Client side. NET Core API for the back end side.

In order to use the application user can enter numeric values on the numeric textbox or drag the pointer on the tempearature. This will automatically convert.

Pleast note that minimum for Celsius is -30 and maximum value is 200. While for the Farenheight minimum is -20 and maximum is 400
